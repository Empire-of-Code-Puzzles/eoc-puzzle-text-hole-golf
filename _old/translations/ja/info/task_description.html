- 匿名の作家

もしあなたが本のページでゴルフをプレイしたかったら、最初にプロットホールを探す必要があります。やってみましょう。

あなたは文字列の配列としてのテキストの塊が与えられます。私達がそれらの行を与えられた通りに配置すると、 様々な列の長さの2次元配列を得ます。 プロットホールは文字や数字や句読点で(上下左右と斜めに)囲まれた単一の空白です。 2つ以上のスペースはプロットホールにはなりません。 空白のすべての「外側」の文字に空白があったらプロットホールにはなりません。 あなたのゴールは与えられたテキストにいくつのプロットホールがあるか数えることです。

例を見てみましょう。 

How are you doing?
I'm fine. OK.
Lorem Ipsum?
Of course!!!
1234567890
0        0
1234567890
Fine! good buy!

ここには3つのプロットホールがあります。"OK"の前、"Lorem"と"Ipsum"の間、"Of course!!!"の中。

このミッションでは主なゴールはあなたのコードを出来る限り短くすることです。 あなたのコードが短ければ、より多くのポイントを獲得することができます。 このミッションでのあなたのスコアは動的であなたのコードの長さに直結しています。 参考までにスコアは使われた文字の数に基づいています。 350文字が許容される最大文字数でこの時は0点です。 350文字から1文字減るごとに1ポイント獲得することができます。 例えば300文字の長さのコードでは50ポイント獲得することになります。

入力 テキストが文字列(py2 では unicode)のリストとして与えられる

出力 単一のホールの数を整数として返す

例 

golf(["How are you doing?",
      "I'm fine. OK.",
      "Lorem Ipsum?",
      "Of course!!!",
      "1234567890",
      "0        0",
      "1234567890",
      "Fine! good buy!"]) == 3
